# 四时有序 · 六爻解卦

一款纯前端的六爻纳甲解卦工具，支持规则引擎自动分析与 AI 研判。

## 在线使用

**https://junxuezha1.github.io/do-not-behind-this-world/**

手机浏览器打开后，点击「添加到主屏幕」可像 APP 一样使用。

## 功能特性

- **卦盘解析** — 粘贴灵光象吉等排盘软件的输出文本，自动识别解析
- **规则引擎** — 自动分析空亡、月建、日辰、动变、旺衰、进退神等
- **AI 研判** — 接入 OpenAI / Claude / DeepSeek 等大模型进行专业断卦
- **应期推断** — 基于空亡、动变、合冲等规则推算应期线索
- **归档系统** — 本地存档分析过的卦例，支持导出 JSON
- **干支日历** — 显示每日天干地支与旬空，方便择日参考
- **五行配色** — 爻线、六神、地支等均按五行着色（木绿火红土黄金白水蓝）
- **PWA 支持** — 可离线使用，支持安装到手机主屏幕

## 使用方法

1. 从灵光象吉等排盘软件复制卦盘文本
2. 粘贴到输入框，点击「解析卦盘」
3. 查看基本信息、规则分析、应期等标签页
4. 切换到「AI 研判」标签页，配置 API Key 后点击「AI 断卦」
5. 点击「归档此卦」保存分析结果

## AI 模型配置

点击右上角「设置」，支持以下模型：

| 模型 | Provider 选择 | Base URL |
|------|-------------|----------|
| OpenAI GPT | OpenAI | （留空用默认） |
| Claude | Claude | （留空用默认） |
| DeepSeek | DeepSeek | （留空用默认） |
| 中转站 | OpenAI 或 自定义 | `https://中转站域名/v1/chat/completions` |

## 技术栈

纯前端，无框架，无构建工具：

- HTML + CSS + JavaScript
- localStorage 存储配置与归档
- Service Worker 实现离线缓存
- SSE 流式输出 AI 分析结果

## 本地开发

双击 `启动服务器.bat`，或手动运行：

```bash
python -m http.server 8080
# 或
npx serve
```
